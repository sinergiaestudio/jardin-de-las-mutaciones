# Jardín de las Mutaciones

Aplicación interactiva basada en el I Ching.

## Descripción
“Jardín de las Mutaciones” propone una experiencia digital de consulta simbólica y reflexión estructurada, inspirada en la tradición del Libro de las Mutaciones.

## Características
- Tirada en modo rápido, ritual y manual
- Lectura de hexagrama principal
- Lectura de hexagrama resultante
- Líneas mutantes
- Biblioteca integrada de los 64 hexagramas
- Diario local de consultas
